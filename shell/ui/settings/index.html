        API keys

        Keys that apps use to call Maximal. Add one per tool so you can tell them apart — anything not listed still works by default.

        Models

        Models available to apps through Maximal, grouped by kind. The list is fetched from the provider — refresh to pull the current catalog if a model looks missing.

        Logs

        Daily-rotated request and handler logs, 7-day retention.

        Where logs live

        Each handler writes a per-day file under the maximal state directory. Reveal the folder to inspect them, or tail the active file with tail -F.

        macOS / Linux
        ~/.local/share/maximal/logs/
        Windows
        %USERPROFILE%\.local\share\maximal\logs\
        Retention
        7 days, then deleted on next boot
