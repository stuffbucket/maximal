Uninstall

Trashing Maximal.app only removes the app bundle. To fully clean up — the launchd agent, the ~/.local/bin/maximal symlink, and (optionally) Claude Desktop’s gateway keys — quit Maximal, then run this in Terminal:

Command
maximal uninstall --revert-claude

Add --purge to also delete stored secrets and config under ~/.local/share/maximal/. Omit --revert-claude to leave Claude Desktop’s config alone.
